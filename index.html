Louise Madden

Hi!

My name is Louise Madden. I’m 20 years old and I have just finished my second year of Computer Games Development in University of Limerick. I chose to do this course due to a workshop I attended with my class in 6th year of Secondary School. We had a short Java workshop and I found that I really enjoyed it. I then discovered the near endless possibilities available when you go on and learn more and learn more languages. It interested me immediately and I have grown to enjoy it even more. Throughout my 2 years in UL I have learned a variety of languages for different purposes. I have enjoyed exploring the new possibilities with each language and the environment they are used in, such as the different versions of Lua used in Corona and PICO-8.

I have joined several societies including the Games society, Games Development Society, WiSTEM2D(Women in Science, Technology, Engineering, Mathematics, Manufacturing and Design) and briefly the Computer Society. These societies allow me to unwind and enjoy some of my hobbies(Games) and expand my knowledge(Game Development). I am currently the Treasurer of the Game Development Society and I hope to run for the position again in September.

My Skills

Java

Java is a very useful and versatile language. It's the first programming language that I learned. I have had a lot of practical experience with the language and have written many programs in Java for projects and labs. It is also the language that introduced me to programming and made me realise that this is what I wanted to study.

C/C++

I have had many modules that require C and/or C++, including Operating Systems, Computer Graphics and Data Structures and Algorithms. I have a basic knowledge of C and a more in-depth knowledge of C++. I have used QT creator to create a basic text-based adventure game. I enjoy using C++ as it has a versatile language that allows for many different types of programs and is also generally a fast and efficient language if used well.

MySQL

I learned basic MYSQL for my Database Systems module. In this module we learned about tables, creating, adding, editing and deleting both tables themselves and the data inserted into them. We also learned constraints and indexes, as well as triggers, functions and procedures. I enjoyed using MYSQL as it was very organised and efficient for quickly finding information.

HTML/CSS

I learned basic HTML and CSS by writing this website. I am teaching it to myself with the help of online guides...and a lot of trial and error. I hope to keep experimenting and learning more of oth of these languages. It is a great way to practise programming and also be creative while exploring options with layouts and colours.

Teamwork
